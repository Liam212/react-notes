## React-notes

This is a simple note taking app that intergrates with google firebase to store notes. User authentication is coming soon.

### `npm install`

This will install all the required packages to run the app.

### `npm start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br />
You will also see any lint errors in the console.
